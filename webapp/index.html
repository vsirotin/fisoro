Welcome in the communist number store!

Why is this store communist? Because you get goods for free in this shop! (Of course it is a joke.)

This page demonstrates browser-side search without any load on the server. All your searches run entirely in your browser.

This is possible thanks to free open source software called FiSoro.

If you want to use FiSoro or learn more about this software - see downloads and details in this GitHub repository: https://github.com/vsirotin/fisoro

You can select each number in interval 0-999. Your number can be represented as x =a*100+b*10+c

You can set properties of a, b, c and x itself. You can restrict x of the digits sum y=a+b+c

You can also set some substring that must appears in text representation of your number.

Have fun as you select and own your numbers!
